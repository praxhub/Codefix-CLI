# CODEFIX-CLI — Local AI-Assisted Python Debugger

CODEFIX-CLI is a lightweight, local-first debugging assistant that helps you analyze Python scripts, detect issues, and generate meaningful fix suggestions. It combines deterministic AST analysis with rule-based patching and optional language model integration. No cloud requirement, no external dependencies unless you explicitly enable them.

## Features

- Local AST-based static analysis  
- Optional LLM-assisted patching (fully offline compatible)  
- Sandbox execution for safe testing  
- Clear explanations of detected issues  
- Clean CLI design, simple commands  
- Suitable for small–medium Python debugging
